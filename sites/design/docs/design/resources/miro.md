# Miro - Visual Collaboration Platform
**Miro** is a versatile **online visual collaboration platform** designed to help teams brainstorm, plan, and manage projects on an infinite digital canvas. 

1. Go to [au.miro.com](https://au.miro.com) and create a free account.
2. Create a board and put in cards for the topics that you've covered so far with the Edisons.
3. Share your board with a classmate.
4. Create another board, use the flowchart template.
5. Create a flowchart showing how you would make an Edison move in a square.
