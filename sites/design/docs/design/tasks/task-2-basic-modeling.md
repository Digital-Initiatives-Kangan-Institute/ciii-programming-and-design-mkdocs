# Task 2 - Basic 3D Modelling

## Basic Keychain
!!! abstract "Instructions"
    Create a basic keychain tag with _your name_. The tag should be at least `5cm` wide and `5mm` thick. Length will be dependent on your name.

## Dice Cube
!!! abstract "Instructions"
    Create a dice cube. The dice should be `5cm` in length. The dots for the dice need to be holes that are `3mm` in depth. **Remember that the number of dots on opposite sides should add up to seven.**
